# transmission-arm-static

This repository contains a build script (`transmission-arm-static.sh`) for compiling **Transmission** as a statically linked executable for ARMv7 Linux devices.

---

## Setup Instructions

1. **Clone this repository**

   ```bash
   git clone https://github.com/solartracker/transmission-arm-static
   cd transmission-arm-static
   ```

2. **Run the build script**

   ```bash
   ./transmission-arm-static.sh
   ```
